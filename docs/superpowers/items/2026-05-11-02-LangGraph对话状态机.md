## 拆解思路

当前后端 `/ws/chat` 为纯 echo，前端聊天 UI 全部就绪但无法与 AI 交互。最小可运行步骤：创建 LangGraph StateGraph + GLM-5 聊天节点，替换 echo 端点为流式 AI 响应。先做单节点原型验证流式对接和响应质量，再添加出题工具函数和 ReAct 条件路由，让智能体根据用户意图调度出题管线。最终清理 echo 残留。
