# 功能项：知识点列表展示

**来源:** [roadmap](../../docs/superpowers/plans/2026-04-29-ai-question-agent.md)
**创建:** 2026-05-12
**状态:** 完成

## 拆解思路

已有知识抽取管线和聊天界面，但提取的知识点只在 WS 文本消息中以 JSON 原文展示。核心缺口：ToolMessage 中的结构化数据未传递给前端。通过在 WS 流中发送 type=data 消息 + 前端解析渲染卡片，打通"聊天中提取→可视化展示"端到端。

## 功能项

### F-1: WS 结构化数据消息
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** WS 流式端点拦截 ToolMessage，发送 type=data 消息给前端
- **验证:**
  - extract_knowledge_points 的 ToolMessage 触发 data 消息
  - generate_questions 的 ToolMessage 触发 data 消息
  - 未知工具的 ToolMessage 不触发 data 消息
- **依赖:** 无
- **代码量:** ~15 行

- [x] **状态:** 已完成
- **生命周期:** 交付

  - 标签颜色按类别区分
- **依赖:** 无
