# 功能项：难度等级控制（基础/中等/提高）

**来源:** [roadmap](../../docs/superpowers/plans/2026-04-29-ai-question-agent.md)
**创建:** 2026-05-12
**状态:** 完成

## 拆解思路

当前题目生成只产出 intermediate 难度。需要将 DifficultyLevel (basic/intermediate/advanced) 串通全链路：模型定义 → LLM prompt 注入 → API 参数 → 前端选项。已有 QuestionStem 模型和 generate_question_llm 函数，F-1 在模型层加字段，F-2 在 LLM 层注入难度指令，F-3 在 API 层暴露参数，F-4 验证端到端。

## 功能项

- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** 在 QuestionStem 中添加 difficulty 字段和 DifficultyLevel 类型

  - DifficultyLevel 接受 "basic"/"intermediate"/"advanced"
- **依赖:** 无
