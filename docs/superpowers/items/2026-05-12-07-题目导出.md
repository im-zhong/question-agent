# 功能项：题目导出

**来源:** [roadmap](../../docs/superpowers/plans/2026-04-29-ai-question-agent.md)
**创建:** 2026-05-12
**状态:** 完成

## 拆解思路

题目数据已在 QuestionStem 模型中完整（含 correct_answer、difficulty），需提供导出能力让用户获取格式化文本。最简方案：后端 Markdown 格式化 + 前端下载按钮。

## 功能项

### F-1: Markdown 导出端点
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** POST /questions/export 接受 questions 数组 + format 参数，返回 Markdown 文件
- **验证:**
  - format=markdown 返回 text/markdown + Content-Disposition 附件头
  - 成功题目包含题干、选项（正确答案 ✓）、知识点、难度、类型
  - 失败题目跳过不导出
- **依赖:** 无
- **代码量:** ~60 行

### F-2: 前端导出按钮
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** 上传页面结果区添加"导出 Markdown"按钮，触发下载
- **验证:**
  - 有成功题目时显示按钮
  - 点击触发浏览器下载 questions.md
- **依赖:** F-1
- **代码量:** ~25 行

### F-3: 测试
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** 4 个测试覆盖 Markdown 导出核心场景
- **验证:**
  - Markdown 导出含正确内容
  - 失败题目不出现
  - JSON 默认格式返回
  - Content-Disposition 头正确
- **依赖:** F-1
- **代码量:** ~80 行
