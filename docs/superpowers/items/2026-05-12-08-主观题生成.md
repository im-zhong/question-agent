# 功能项：主观题生成（简答/论述）

**来源:** [roadmap](../../docs/superpowers/plans/2026-04-29-ai-question-agent.md)
**创建:** 2026-05-12
**状态:** 完成

## 拆解思路

已有选择题全链路。主观题（简答/论述）核心差异：无选项，有参考答案 reference_answer。需新增 prompt 模板、扩展数据模型、更新 API 参数、适配前端卡片渲染。

## 功能项

### F-1: 数据模型扩展
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** QuestionType 新增 short_answer/essay，QuestionStem 新增 reference_answer 字段
- **验证:**
  - short_answer 和 essay 是合法 QuestionType
  - reference_answer 默认 None，主观题时必填
- **依赖:** 无
- **代码量:** ~10 行

### F-2: 主观题 prompt 模板
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** 新增 _SHORT_ANSWER_PROMPT 和 _ESSAY_PROMPT，含 reference_answer schema + 示例
- **验证:**
  - 两个模板均含 reference_answer 字段
  - select_prompt(question_type="short_answer") 返回对应模板
  - question_type 优先级高于 tags 类别
- **依赖:** F-1
- **代码量:** ~50 行

### F-3: LLM + Generator 层适配
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** generate_question_llm 接受 question_type 参数，_LlmQuestionResult 支持 reference_answer
- **验证:**
  - question_type="short_answer" 时使用主观题 prompt
  - 返回的 QuestionStem 含 reference_answer，options=[]
- **依赖:** F-2
- **代码量:** ~30 行

### F-4: API + 智能体调度适配
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** API 端点接受 question_type 参数，dispatch_node 从用户消息检测题型
- **验证:**
  - POST /questions/generate 接受 question_type 字段
  - "出简答题" 触发 short_answer，"出论述题" 触发 essay
- **依赖:** F-3
- **代码量:** ~25 行

### F-5: 前端主观题卡片渲染
- [x] **状态:** 已完成
- **生命周期:** 交付
- **目标:** 主观题不显示选项，显示参考答案折叠区
- **验证:**
  - short_answer/essay 题型不渲染选项列表
  - 参考答案以虚线边框区域展示
  - 题型徽章颜色区分（cyan/indigo）
- **依赖:** F-3
- **代码量:** ~40 行
