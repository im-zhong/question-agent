# 功能项：对话历史持久化

**来源:** [roadmap](../../plans/2026-04-29-ai-question-agent.md)
**创建:** 2026-05-12
**状态:** 完成

## 拆解思路

当前 MemorySaver 是内存级 checkpoint，进程重启丢失所有对话。thread_id 每次新建 WS 连接生成 UUID4，页面刷新即创建新会话。需要：SqliteSaver 替换 MemorySaver 实现跨重启持久化；稳定 conversation_id 使同一对话可跨连接恢复；WS 连接时回放历史消息；前端 localStorage 存 conversation_id 实现刷新恢复。先替换 checkpointer 验证持久化生效，再做 ID 稳定化和前端恢复。
