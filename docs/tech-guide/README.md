- **快速了解全局**：先读 01（概览），再读 02（Workflow）
- **理解 Agent 机制**：重点读 05（LangGraph Agent），配合 03（混合检测）
- **深入某个模块**：直接跳转到对应文档
- **开发接口对接**：参考 08（API 参考）
